# 🎯 WPF Quiz Application (Kahoot-like)

This project is a **WPF desktop application** for creating and hosting quizzes, similar to **Kahoot**.  
It was developed as a **university project** by students **Fuad, Yulia, and Susanna**.

## 📌 Project Overview

The application allows users to create, save, load, and play quizzes in an interactive way.  
It supports multiple languages, sound effects, and quiz customization options.

All requirements of the course were fully implemented in the final version of the project.

## 🖥️ Main Features

### 🏠 Main Screen
On the main screen, the user can:
- Start a new quiz
- Load an existing quiz from the database
- Exit the application

The main screen also allows the user to select one of **three available languages**.

### ✍️ Quiz Creation
In the quiz creation screen, the user can:
- Write quiz questions
- Add multiple answer options
- Select the type of answer
- Set a time limit for answering each question

### 🎮 Quiz Gameplay
- Questions are shown one by one
- Each question has a countdown timer
- Sound effects are used to improve user experience

### 📊 Results & Saving
- At the end of the quiz, the user can see their results
- The application offers an option to save the created quiz for later use

## 🌍 Localization
- The application supports **three languages**
- Localization is implemented using **resource files**

## 🔊 Additional Features
- Sound effects during gameplay
- Structured and user-friendly interface
- Data storage for saving and loading quizzes
